# HISTORIA DE ARDUINO
Es una plataforma de hardware basada en un microcontrolador principalmente Atmel AVR montado en una PCB(Printed Circuit Board o Placa de Circuito Impreso) con los elementos esenciales para su funcionamiento pensado para proyectos electricos multidisciplinarias. 
Fue creada en 2005 en Italia en el Interaction Design Institute Ivrea con el objetivo de facilitar el aprendizaje de electrónica y programación a estudiantes y diseñadores, sus principales creadores fueron:
1. Massimo Banzi
2. David Cuartielles
3. Tom Igoe
4. Gianluca Martino
5. David A. Mellis
