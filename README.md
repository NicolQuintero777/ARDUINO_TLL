# HISTORIA DE ARDUINO
Es una plataforma de hardware basada en un microcontrolador principalmente Atmel AVR montado en una PCB(Printed Circuit Board o Placa de Circuito Impreso) con los elementos esenciales para su funcionamiento pensado para proyectos electricos multidisciplinarias. 
Fue creada en 2005 en Italia en el Interaction Design Institute Ivrea con el objetivo de facilitar el aprendizaje de electrónica y programación a estudiantes y diseñadores, el estudiante colombiano Hernando Barragán, fue quien desarrolló la tarjeta electrónica Wiring,el lenguaje de programación y la plataforma de desarrollo sus principales creadores que aportaron fueron:
1. Massimo Banzi
2. David Cuartielles
3. Tom Igoe
4. Gianluca Martino
5. David A. Mellis

# PERIFERICOS DE ARDUINO Y SU USO
Arduino puede interactuar con múltiples periféricos que permiten ampliar sus funcionalidades:

## Sensores
Permiten obtener información del entorno.
1. Sensor de temperatura (LM35)
2. Sensor de distancia (ultrasonido)
3. Sensor de luz (LDR)

## Actuadores
Ejecutan acciones físicas.
1. Motores (DC, servomotores)
2. Relés
3. LEDs

## Módulos de comunicación
Permiten conectar Arduino con otros dispositivos.
1. Bluetooth (HC-05)
2. WiFi (ESP8266)
3. RF (radiofrecuencia)

## Pantallas
Permiten mostrar información.
1. LCD
2. OLED

# FICHA TECNICA, PINES DE CONEXION
1. Microcontrolador: ATmega328P
2. Voltaje de operación: 5V
3. Voltaje de entrada: 7V - 12V
4. Pines digitales: 14 (6 PWM)
5.Pines analógicos: 6
6. Memoria Flash: 32 KB
7. SRAM: 2 KB
8. EEPROM: 1 KB
9. Frecuencia: 16 MHz
